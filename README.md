# Mail Panic

## Description 

This project is made to simulate a day at a mail office, with a very punctual employee.

## Languages 

- Python

## Requirements

- Python 3.9+
- Python's random library

## Installation

1. Clone repository
```cmd
    git clone https://github.com/viktortle/mail_panic
```
